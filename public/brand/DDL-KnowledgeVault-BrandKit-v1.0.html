07 — Usage Rules
Do / Don't
Do
  • Use the full seal at 48px and above
  • Use KV monogram-only at 32px and below
  • Use on navy, card, dark, cream, or teal backgrounds
  • Maintain the teal ring and cream/navy monogram contrast
  • Use the wordmark lockup for UI sidebars and headers
  • Use the canon/archive badge variants in the ingestion pipeline
  • Pair with Space Grotesk for all display and UI text
  • Use the steel (#6B7B8D) for all secondary metadata
Don't
  • Place on white backgrounds (CottageHumble: no white)
  • Use Inter, Roboto, or system fonts anywhere
  • Change the teal from #2C7A7B without ratification
  • Stretch or distort the seal proportions
  • Use the vault teal for DDL, D&A, DexVerse, or Dossier contexts
  • Add drop shadows, glows, or gradient overlays to the mark
  • Use the full seal below 48px — use the reduction mark instead
  • Place the mark on busy backgrounds without contrast clearance

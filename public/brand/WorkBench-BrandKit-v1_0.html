01 — Primary Mark
The Module Grid

The WorkBench mark is a 3×2 grid of modules — some filled (active), some outlined (available). The asymmetry is intentional: you choose what you need. The bottom bar communicates progress and coverage. Scales from favicon to hero.

06 — Module Cards
Pick and Choose Interface

The core UI pattern. Each module is a card. Selected modules get a sky border and a focus ring. Available (not selected) modules show a standard border. This pattern applies to both the marketing page and the in-app selector.

Selected cards have sky border + sky-lt ring. Available cards have standard var(--border).

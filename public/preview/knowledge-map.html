Governance Layer

Data Layer
SQLite + WAL
4 tables. artifacts fact table with 21 fields. ingestion_log audit trail. knowledge_gaps reserved for Phase 4. FK enforcement per connection. CHECK constraints on generation_method, status, confidence, lineage_depth.
Logic Layer
ACE Wrapper
governance.py — the gate everything passes through. Create, update, review, archive, search. SHA-256 content hash for dedup. UUID artifact IDs with dex- prefix. Lineage tracking via parent_artifact_id.
Interface Layer
CLI Pipeline
python -m dexverse.ingest — one command, file or directory. python -m dexverse.registry — list, search, stats, show, review, archive, log. Full artifact lifecycle from terminal.
Governance Layer
13 Invariant Tests
Adversarially reviewed by Grey (ChatGPT) before lock. Every constraint proven to enforce, not just define. FK active, WAL active, CHECK constraints fire, parent validation fails fast, dedup catches duplicates, lifecycle promotes correctly.

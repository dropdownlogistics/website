# Dropdown Logistics

**Chaos → Structured → Automated**

The public-facing surface of DDL — a one-person operations studio building governed systems, dimensional architectures, and automation frameworks.

## What's Here

- **44 Governed Systems** — filterable registry with full records
- **65 Enforced Standards** — OBS/PRO/REC/VEH classifications
- **Excelligence Graph Explorer** — force-directed knowledge graph
- **Foreword Convergence** — nine AI models, one thesis
- **PrioritEase v2** — interactive priority × ease scoring
- **DDL Council** — nine-model convergent analysis methodology
- **Little to Know Experience** — memoir, weekly on Substack

## Tech Stack

- Next.js 14 (React, static site generation)
- Tailwind CSS
- TypeScript
- Deployed on Vercel
